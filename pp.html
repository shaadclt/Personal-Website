PalatePixels: Image-based Recipe Retrieval System

  • Github: https://github.com/shaadclt/Image-based-Recipe-Retrieval-System

• Primary Goal : To create a user-friendly platform that enables effortless recognition of Indian recipes from images while offering detailed recipe information .
• Employed deep learning models, specifically DenseNet201, for image encoding and feature extraction.
• Utilized web scraping techniques with Beautiful Soup to collect and organize data on Indian recipes.
• Developed a user-friendly web application using the Django framework to provide a seamless recipe recognition and information platform.
